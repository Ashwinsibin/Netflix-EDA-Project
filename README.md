# Netflix EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Tasks Performed
- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Business Insights

## Visualizations
- Bar Chart
- Line Chart
- Histogram
- Scatter Plot
- Pie Chart
- Heatmap

## Dataset
Netflix Movies and TV Shows Dataset (Kaggle)
